\## Thetis N4EAC Edition RC2

A minor modification to include FFT Max Bin signal reading to the legacy meter and the
removal of 6m and VHF+ band buttons for HL2. Thanks & credits to MI0BOT.

RC2 is a packaging and presentation release.

\### RC2 Changes



\- Added a Windows installer using Inno Setup.

\- Added automatic installation of the N4EAC Edition RC2 splash screen.

\- Added SDR-themed installer, shortcut, uninstall, and application window icons.

\- Added custom installer wizard images.

\- Added GPL license page to the installer.

\- Fixed Windows 11 build-date generation by replacing deprecated WMIC usage with PowerShell.



No DSP, CAT, meter, or radio-control functionality was changed in RC2.
