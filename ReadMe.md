Thetis N4EAC Edition

A customized version of the OpenHPSDR Thetis software optimized for the Hermes Lite 2 (HL2) SDR transceiver.

This project preserves the excellent functionality of the original Thetis software while introducing practical enhancements, usability improvements, and features tailored to HF operation.

Current Release

Thetis N4EAC Edition RC1

Building

Requirements:

Microsoft Visual Studio 2022
.NET Framework (same version required by the original Thetis project)
WDSP libraries included with the source
Original project dependencies

Open the solution in Visual Studio and build the Release x64 configuration.

Project Goals

This project focuses on:

Hermes Lite 2 usability
Enhanced metering capabilities
Cleaner user interface
Maintaining compatibility with the original Thetis architecture whenever practical
