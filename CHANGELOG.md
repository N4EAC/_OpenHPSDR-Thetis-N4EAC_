# Thetis N4EAC Edition Changelog

All notable changes to the N4EAC Edition are documented in this file.

The N4EAC Edition is based on the OpenHPSDR Thetis project and is distributed under the terms of the GNU General Public License (GPL).

# RC2

Release Date: July 2026

## Added

- Professional Windows installer using Inno Setup.
- Automatic installation of the N4EAC Edition splash screen.
- Custom installer icon.
- Custom installer welcome artwork.
- Custom Start Menu shortcut icon.
- Custom Desktop shortcut icon.
- Custom application window icon.
- GPL license page displayed during installation.

## Changed

- Updated application branding to **N4EAC Edition RC2**.
- Modernized Windows build process by replacing the deprecated **WMIC** dependency with **PowerShell** for automatic build date generation.
- Improved installation experience with automated splash screen deployment.
- Improved installer presentation and visual branding.

## Fixed

- Corrected Windows 11 build-date generation.
- Corrected installer packaging to include all required branding assets.
- Corrected icon consistency across installer, shortcuts, taskbar, uninstall entry, and application window.

---

# RC1

Release Date: 2026

## Added

- FFT Maximum Bin signal level support for Legacy RX1 meter.
- FFT Maximum Bin signal level support for Legacy RX2 meter.

## Changed

- Simplified Hermes Lite 2 band button layout by hiding unused controls.
- Updated splash screen branding.
- Updated About dialog branding.

## Notes

RC1 focused on functional improvements for Hermes Lite 2 users while maintaining compatibility with the upstream Thetis project.
